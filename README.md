## 🛠️ Especificaciones Técnicas del Reto
El proyecto está estructurado de manera limpia y modular, utilizando las mejores prácticas de desarrollo frontend básico:

*   **`index.html`** ➔ Estructura semántica organizada por bloques funcionales.
*   **`styles.css`** ➔ Diseño moderno con Grid, Flexbox, sombras personalizadas y bordes dinámicos.
*   **`script.js`** ➔ Interactividad avanzada a través del DOM (Eventos click y un Reloj Digital en tiempo real).
*   **Recursos Locales** ➔ Optimización de carga utilizando imágenes locales exclusivas en formato PNG.
